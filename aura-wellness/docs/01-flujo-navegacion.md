# Flujo de Navegación - Aura Wellness

## 📋 Descripción General

El flujo de navegación de Aura Wellness está diseñado para ser intuitivo y eficiente, priorizando la simplicidad y la accesibilidad. La aplicación sigue un patrón de navegación jerárquica con un drawer lateral colapsable.

## 🗺️ Estructura de Navegación

### Navegación Principal (Bottom Navigation)
- **Inicio**: Dashboard con métricas principales
- **Métricas**: Vista detallada de salud con carousel
- **Agregar**: Registro rápido de actividades
- **Perfil**: Configuración y datos personales

### Navegación Secundaria (Drawer)
- **Principal**
  - Inicio
  - Métricas
  - Objetivos
  - Progreso
- **Salud**
  - Ritmo Cardíaco
  - Sueño
  - Hidratación
  - Actividad Física
- **Configuración**
  - Ajustes
  - Notificaciones
  - Modo Oscuro

## 🔄 Flujos de Usuario

### Flujo de Primer Uso
1. Pantalla de Bienvenida
2. Configuración Inicial
3. Conexión de Dispositivos
4. Dashboard Principal

### Flujo Diario Típico
1. Dashboard → Ver métricas principales
2. Métricas → Revisar detalles específicos
3. Agregar → Registrar nueva actividad
4. Perfil → Ajustar configuración si necesario

### Flujo de Configuración
1. Perfil → Configuración
2. Ajustes → Modificar preferencias
3. Notificaciones → Gestionar alertas
4. Privacidad → Controlar datos

## 🎯 Jerarquía Visual

### Nivel 1: Pantallas Principales
- Alto contraste, elementos grandes
- Información esencial visible inmediatamente

### Nivel 2: Pantallas de Detalle
- Información contextual adicional
- Navegación clara hacia atrás

### Nivel 3: Configuración y Ajustes
- Elementos más densos
- Funcionalidad específica

## 📱 Patrones de Interacción

### Gestos
- **Swipe horizontal**: Navegar entre métricas
- **Swipe desde borde**: Abrir drawer
- **Tap**: Seleccionar elementos
- **Long press**: Acciones contextuales

### Estados
- **Loading**: Indicadores de progreso
- **Error**: Mensajes claros con acciones
- **Success**: Confirmaciones positivas
- **Empty**: Estados vacíos con CTAs

## ♿ Accesibilidad

### Navegación por Teclado
- Tab order lógico
- Skip links para navegación rápida
- Focus indicators visibles

### Soporte de Lectores de Pantalla
- Labels descriptivos
- Landmark regions
- Live regions para actualizaciones

### Alto Contraste
- Cumple WCAG AA (4.5:1)
- Modo oscuro opcional
- Colores accesibles

## 📊 Métricas de Usabilidad

### Objetivos
- **Task Completion**: >95%
- **Time to Complete**: <30 segundos para tareas comunes
- **Error Rate**: <5%
- **User Satisfaction**: >4.5/5

### KPIs de Navegación
- **Bounce Rate**: <20%
- **Session Duration**: >5 minutos
- **Feature Adoption**: >70% para funciones principales

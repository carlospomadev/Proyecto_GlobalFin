# 📋 README - Aura Wellness

> **Un enfoque de simplicidad y accesibilidad en bienestar digital**

## 🎯 Propósito del Proyecto

Aura Wellness es una aplicación móvil multiplataforma diseñada para simplificar el seguimiento de métricas de salud y bienestar. En un mercado saturado de aplicaciones complejas, Aura se destaca por priorizar:

- **Simplicidad Visual**: Jerarquía clara, sin información innecesaria
- **Accesibilidad**: WCAG AA compliance, soporte para diversos usuarios
- **Experiencia Intuitiva**: Gestos naturales, menús colapsables, flujo sin fricción
- **Diseño Inclusivo**: Colores accesibles, navegación por teclado, lectores de pantalla

## 📱 Plataformas Soportadas

```
✓ iOS (13+)
✓ Android (6.0+)
✓ Web (Progressive Enhancement)
```

## 🏗️ Stack Tecnológico

- **Framework**: Flutter 3.x
- **State Management**: BLoC Pattern
- **Arquitectura**: Clean Architecture
- **Base de datos**: Hive (local), REST API (remoto)
- **Idioma**: Dart 3.x
- **Design System**: Material Design 3

## 🎨 Estrategia de Diseño

### Colores Principales
- **Primary**: Verde (#4CAF50) - Salud y naturaleza
- **Secondary**: Azul (#2196F3) - Confianza y calma
- **Tertiary**: Ámbar (#FF9800) - Energía y motivación

### Tipografía
- **Fuente Principal**: Inter (Google Fonts)
- **Escala**: Material Design 3 Typography Scale

### Espaciado
- **Base**: 4px (0.25rem)
- **Escala**: 4, 8, 12, 16, 24, 32, 48, 64px

### Accesibilidad
- **Contraste**: Mínimo 4.5:1 para texto normal
- **Tamaño Mínimo**: 44px para elementos táctiles
- **Soporte**: Screen readers, navegación por teclado
